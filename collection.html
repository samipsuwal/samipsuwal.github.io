Books
CSS:The Missing Manual

A very good book that goes over various concepts of CSS, and provides good explanation on them. The book starts with basic concepts and ventures into some advance topics in CSS. This book should be read by all levels of readers, however beginners and intermediate will benefit the most. Book is in touch with what is being used in web and gives examples that can be seen in real web pages. This book has a very good layout where it will walk you through couple of exercises. For me its super important because I need to actually work on something to remember it.

Pro Git

This book is the encyclopedia of GIT. This book covers pretty much every topic for every level and type of GIT users. From Junior to Senior programmers to System architects, this book will serve them all very well. The book is free in the official GIT website, which sort of makes it their official book. There are 10 chapters in the book. However, you don’t really have to read the entire book. You can cherry pick chapters depending on you need. As a software engineer I found chapter 2nd, 3rd, and 7th extremely useful.
